## Запуск проекта
Запуск по шагам из корневой директории:

0. Установить зависимости проекта: `npm i`
1. Запустить клиент и сервер: `npm run start`.

2. Запуск тестов проекта: `npm test`

3. Сборка проекта `npm run build`
